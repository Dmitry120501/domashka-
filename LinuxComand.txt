Каталог = папка.

Существует абсолютный и относительный адресс.
Абсолютный - это адресс, который начинается с корневого каталога.
Относительный - это адресс, который пишется относиельно каталога в котором ты находишься.

./ - адресс каталога (папки) в котором ты находишься.
/ - адресс корневого каталога.
~ - адресс домашнего каталога.
../ - адрес каталога на уровень выше.

1) Как перейти в домашний каталог из любого другого?
cd ~ (переход в домашний каталог)
cd / (переход в корневой каталог)
cd ../ (переход на каталог ниже)

2) Как создать новый текстовый файл 123.txt  в текущем каталоге?
nano 123.txt (просто создание файла)
touch 123.txt (также применяется для обновления метов времени файла)
> 123.txt (таже применяется для перезаписывания файла)
cat > 123.txt (также применяется для чтения и объединения файлов)
echo "text" > 123.txt (можно сразу написать текст, который будет в файле)

3) Как создать новую папку 1234 в текущем каталоге?
mkdir 1234

4) Как создать в каталоге /abc (для MacOS /Library/User Pictures/Animals/abc) ссылку на файл 123.txt из домашнего каталога?
ln ~/123.txt abc

5) Как создать в каталоге /abc (для MacOS /Library/User Pictures/Animals/abc) символьную ссылку на папку 1234 из домашнего каталога?
ln -s ~/1234 abc

6) Как скопировать файл 123.txt из домашнего каталога в корневой?
sudo cp ~/123.txt /
потом вводишь пароль: dunice

7) Как переместить файл 123.txt из домашнего каталога  в каталог на 2 уровня выше?
sudo mv ~/123.txt ../..
потом вводишь пароль: dunice

8) Как переименовать файл 123.txt  в файл txt.123?
mv 123.txt txt.123

9) Как удалить файл 123.txt?
rm 123.txt

10) Как удалить все файлы и папки из каталога /abc (для MacOS /Library/User Pictures/Animals/abc), не удаляя сам каталог?
rm -r abc/*

11) Как удалить все файлы и папки из какого-нибудь каталога без всплывающих вопросов?
rm -rf abc/*

12) Как переместиться в другую папку(abc) родительского каталога (соседнюю по отношению к текущей)?
cd ../abc

13) Как посмотреть в каком каталоге мы сейчас находимся?
pwd

14) Как посмотреть содержимое текущего каталога?
ls

15) Как посмотреть все файлы (включая скрытые) в текущем каталоге?
ls -a

16) Как посмотреть права доступа к файлу? Что такое пользователь и группа в данном контексте?
Пользователь - аккаунт, который имеет или не имеет доступ к каким либо файлам;
Группа - любая группа пользователей, существующая в системе и привязанная к файлу.

ls -l 123.txt

17) Как записываются права и как они расшифровываются?
chmod (команда для работы с правами)

0 - никаких прав;
1, x - права только на выполнение;
2, w - права только на запись (изменение);
4, r - права только на чтение;
3 - права на выполнение и запись (1+2);
5 - права на выполнение и чтение (4+1);
6 - права на выполнение и запись (4+2);
7 - права на выполнение и чтение и запись (4+1+2);

a - абсолютно все пользователи
u - владелец файла;
g - группа файла;
o - все остальные пользователи.

Примеры с цифрами:
744 - разрешить все для владельца, а остальным только чтение;
755 - все для владельца, остальным только чтение и выполнение;
764 - все для владельца, чтение и запись для группы, и только чтение для остальных;
777 - всем разрешено все.

18) Как поменять права файлу, разрешив всем только читать данный файл?
chmod 444 123.txt (ответ на вопрос)

chmod ugo-rwx 123.txt (забираем все права на фаил у всех пользователей);
chmod ugo+r 123.txt (всем пользователям даем права только на чтение данного файла).

19) Как устанавливать новое программное обеспечение на платформе с которой ты работал?
sudo apt-get install name.exe - установка пакета;
sudo apt install name.exe
sudo apt-get remove name.exe - удаление пакета;
sudo apt-get download name.exe - скачать deb-пакет в локальную папку;
sudo apt-get upgrade - обновление всех установленных пакетов до актуальной версии, имеющейся в репозитории;
sudo apt-get update - обновление информации о пакетах и зависимостях в подключенных репозиториях.

sudo dpkg -i name.deb - установка пакета;
sudo dpkg -r name.deb - удаление пакет; (не обязательно)
sudo dpkg -l - вывод списка установленных пакетов. (не обязательно)

Для Мака:
brew innstall name.exe - установка

20) Как посмотреть список процессов, если известна часть имени процесса?
ps -A | grep name
ps aux | grep name

21) Как остановить эти процессы по PID?
kill -9 namePid (SIGKILL) - не может быть проигнорирован, прекращение работы немедленно;
kill -15 namePid (SIGTERM) - может быть проигнорировал, дает время процессу завершиться нормально;
kill -17 namePid (SIGSTOP) - приастоновка работы процесса;
kill -19 namePid (SIGCONT) - продолжение работы процесса;

22) Как скопировать все содержимое из папки /abc (для MacOS /Library/User Pictures/Animals/abc) в папку /bcd (для MacOS /Library/User Pictures/Animals/bcd) (нужно скопировать все содержимое, а не папку целиком)?
cp -r abc/* bcd
